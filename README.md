# 🛒 NovaStore

A modern full-stack e-commerce web application built with **React**, **Node.js**, **Express.js**, **Prisma ORM**, **PostgreSQL (Neon)**, and **JWT Authentication**.

NovaStore provides a seamless online shopping experience where customers can browse products, manage their shopping cart, place orders, and securely access their accounts. It also includes an administrative dashboard for managing products and customer orders.



---

# ✨ Features

* User Registration
* Secure User Login
* JWT Authentication
* Browse Products
* Shopping Cart Management
* Checkout System

* Responsive User Interface

NovaStore was developed to demonstrate practical full-stack software development skills, including modern frontend development, RESTful API design, authentication and authorization, database management with Prisma and PostgreSQL, and secure role-based access control. The project showcases how a complete e-commerce platform can be designed, implemented, and managed using contemporary web technologies.
